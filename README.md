🎓 EduHub – Smart Learning Platform
📌 Project Overview

EduHub is a modern online learning platform designed to connect students and tutors in a seamless and efficient way.
The system allows students to discover tutors, book sessions, and manage learning activities, while tutors can create sessions and track their engagement.

The platform also includes an admin panel to manage users, inquiries, and overall system operations.
